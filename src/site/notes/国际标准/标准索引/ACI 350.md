环境工程混凝土结构规范
Code Requirements for Environmental Engineering Concrete Structures

​​核心内容​​：
​​耐久性设计​​：针对污水厂、水坝等环境严苛结构，提出抗腐蚀、抗渗要求。
​​裂缝控制​​：限制有害裂缝宽度，延长结构使用寿命。

​​适用范围​​：污水处理设施、水利工程等需长期耐久的混凝土结构
